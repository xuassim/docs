## Nouvelle méthode

La nouvelle méthode de connexion est beaucoup plus simple : 

1. Rendez vous dans Tableau de bord>Whatsapp Cloud (ou Nuage Whatsapp)

2. Connectez vous via Facebook

3. Pendant la connexion, suivez toutes les étapes dans la popup pour choisir un "Business Account", choisir ou créér un nouveau compte Whatsapp Business, et vérifier votre numéro de téléphone.

4. Assurez vous d'avoir une méthode de paiement valide reliée à votre compte Whatsapp Business.
