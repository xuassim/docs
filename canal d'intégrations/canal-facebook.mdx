Suivez les étapes 1, 2 et 3 de la capture d'écran ci-dessus et connectez-vous ou inscrivez-vous. Sélectionnez toutes les pages dans votre liste de pages, que vous voulez que LineChat automatise avec un assistant virtuel.

<img height="50" src="https://clickway.fr/wp-content/uploads/2024/03/Untitled-10.png" />

Après la connexion, voici quelques-unes des opérations figurant dans la capture d'écran ci-dessus :

Cliquez sur "Reconnect" lorsque vous souhaitez ajouter/supprimer une page ou actualiser.

Cliquez sur "Create Flow" (Créer un flux) pour créer un nouveau flux pour le robot.

Cliquez sur "Ouvrir le flux" pour entrer dans le flux.

Cliquez sur le x rouge pour supprimer la connexion entre le robot et votre page.

Basculez le statut pour arrêter/démarrer l'automatisation.

Cliquez sur le lien pour parler au robot. Vous pouvez également partager le lien avec d'autres utilisateurs de Facebook.
