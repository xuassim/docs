---
title: 'Calendly'
---
L'intégration de Calendly est un excellent moyen de créer des réservations pour votre entreprise ou celle de vos clients. LineChat propose une intégration native qui vous permet de configurer cela très facilement.

### Connexion à Calendly

Depuis le tableau de bord principal, allez dans les **Intégrations** dans le menu de gauche. Sélectionnez ensuite **Calendly** dans la liste. Appuyez sur le bouton "Connecter compte Calendly" pour aller sur Calendly et autoriser LineChat à gérer les réservations pour vous.

<Info>
Lorsque vous êtes redirigé vers la page d'autorisation, assurez-vous d'appuyer sur le bouton. Vous serez redirigé vers votre compte, qui devrait maintenant être vérifié. Si vous avez un compte professionnel, vous verrez également une coche avec la section webhook.
</Info>

### Utilisation de Calendly

Dès que Calendly est connecté, vous pourrez l'utiliser dans l'un de vos chatbots. Allez dans un flux, sélectionnez un message texte et appliquez un bouton. Pour le type de bouton, vous pouvez sélectionner le bouton **Calendly**.

Une fois cela fait, vous pourrez sélectionner un type d'événement. Ces événements sont récupérés à partir de votre compte Calendly. Lorsque vous envoyez ce message avec le bouton au souscripteur, il ou elle pourra réserver un rendez-vous en utilisant l'événement que vous avez configuré.

### Récupération des détails de la réservation

Après cela, vous pourrez récupérer les détails de la réservation à partir du champ système "Sélectionner". Ouvrez la liste des variables système, recherchez le champ système appelé "Sélectionner". Une fois appuyé, vous obtiendrez une fenêtre contextuelle qui affichera toutes les informations possibles que vous pouvez récupérer à partir de Calendly.
