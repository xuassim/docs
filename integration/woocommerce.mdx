---
title: 'WooCommerce'
---
LineChat est compatible avec WooCommerce, un plugin de commerce électronique open-source pour WordPress. Ce plugin est conçu pour les marchands en ligne de petite à grande taille utilisant WordPress. Il est rapidement devenu populaire en raison de sa facilité d'installation et de personnalisation, ainsi que de sa position de marché en tant que logiciel gratuit.

Aujourd'hui, WooCommerce est l'une des options leaders pour les magasins en ligne et pour les personnes qui souhaitent ouvrir leur propre magasin en ligne et gérer les commandes à partir de celui-ci.

### Intégration native de WooCommerce

LineChat prend en charge l'intégration native de WooCommerce, ainsi que toutes les actions prises en charge par l'API actuelle de WooCommerce dans ses points de terminaison. Vous pouvez non seulement créer des automatisations pour votre magasin WooCommerce à l'aide de LineChat, mais également créer, modifier, mettre à jour des commandes, les livrer, suivre les commandes, les expéditions, les évaluations, etc.

### Étapes d'intégration

Pour intégrer LineChat avec votre magasin WooCommerce, vous devez suivre les étapes suivantes :

### 1. Se connecter à votre magasin WooCommerce

Accédez à WordPress et cliquez sur l'onglet WooCommerce.

### 2. Configurer l'API REST

Cliquez sur Paramètres > Avancé > Onglet API REST.

### 3. Créer une clé API

Click on Add Key et entrez les détails pour créer votre clé API.

### 4. Enregistrer et copier les détails

Enregistrez et copiez les détails de votre clé API.

### 5. Intégrer votre magasin WooCommerce avec LineChat

Sur LineChat, cliquez sur l'onglet Intégrations et cliquez sur WooCommerce pour intégrer votre magasin. Entrez ensuite les différents identifiants que vous venez de créer dans ce même onglet.

### Utilisation des actions WooCommerce

Pour utiliser les actions WooCommerce, créez une carte action, puis cliquez sur Intégrations pour trouver l'onglet WooCommerce et cliquez sur celui-ci. Vous trouverez alors diverses actions WooCommerce que vous pouvez utiliser pour automatiser votre magasin en ligne.

<Info>
Assurez-vous d'avoir suivi les étapes d'intégration pour utiliser les actions WooCommerce.
</Info>
