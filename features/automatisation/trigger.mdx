---
title: 'Déclencheurs'
---

Les déclencheurs peuvent vous aider à automatiser certaines processus dans le chatbot. Par exemple, lorsque un utilisateur a opté pour recevoir des e-mails, vous pourriez vouloir synchroniser ces données avec un CRM ou une feuille de calcul Google.

### Accéder aux déclencheurs

Pour accéder à la section des déclencheurs, vous devez aller dans le menu de gauche et sélectionner "Automatisation", puis choisir "Déclencheurs". À partir de là, vous pourrez créer un nouveau déclencheur en appuyant sur le bouton bleu en haut à droite nommé "+ Nouveau déclencheur". **Vous pouvez également y accéder depuis la carte "Start".**

### Créer un nouveau déclencheur

Prenons l'exemple d'un déclencheur lorsque quelqu'un paie une commande à l'intérieur de votre chatbot. Appuyez sur le bouton bleu "+ Nouveau déclencheur", puis sélectionnez le déclencheur nommé "Commande payée".

À partir de là, une nouvelle fenêtre contextuelle s'ouvrira où vous pourrez configurer les détails de votre déclencheur.

Vous pourrez activer ou désactiver le déclencheur, configurer une condition si vous devez que l'utilisateur du bot corresponde à un filtre spécifique avant que le déclencheur ne soit activé, et vous pouvez ajouter une note qui expliquera en détail ce déclencheur.

C'est utile si vous travaillez avec une équipe ou créez un modèle et devez clarifier les choses.

### Sélectionner votre sous-flux

Maintenant que le déclencheur a été créé, nous devons ajouter une action à celui-ci. Vous pouvez le faire en appuyant sur l'espace "Choisir sous-flux" à droite du déclencheur.

Une fois fait, cela ouvrira une fenêtre contextuelle pour vous où vous pourrez sélectionner votre flux souhaité pour être connecté une fois que ce déclencheur aura été activé.

Vous pouvez également taper pour trouver votre flux plus facilement si vous le préférez. Une fois que vous avez trouvé votre flux, sélectionnez-le et il vous ramènera à la vue d'ensemble du déclencheur.

### Gérer vos déclencheurs

Vous avez deux options si vous voulez gérer vos déclencheurs :

* Vous pouvez activer ou désactiver les déclencheurs en appuyant sur le curseur à gauche du déclencheur réel
* Vous pouvez éditer les paramètres du déclencheur. À l'intérieur de cela, vous avez également l'option de supprimer le déclencheur entièrement

### Glossaire des déclencheurs

#### Nouvel utilisateur

Ce déclencheur est utilisé lorsque nouvel utilisateur du bot entre dans le bot. Ce déclencheur ne fonctionnera pas lorsque l'utilisateur est créé manuellement.

#### Utilisateur abonné au bot

Ce déclencheur se produit lorsque l'utilisateur du bot s'est abonné au bot.

#### L’utilisateur s’est désabonné

Ce déclencheur se produit lorsque l'utilisateur du bot s'est désabonné du bot.

#### L’utilisateur à opté pour l’e-mail

Ce déclencheur se produit lorsque l'utilisateur du bot a opté pour recevoir des e-mails, soit manuellement soit par flux

#### L’utilisateur s’est désabonné de l’e-mail

Ce déclencheur se produit lorsque l'utilisateur du bot a refusé de recevoir des e-mails, soit manuellement soit par flux

#### L’utilisateur a opté pour les SMS

Ce déclencheur se produit lorsque l'utilisateur du bot a opté pour recevoir des SMS, soit manuellement soit par flux

#### L’utilisateur s’est désabonné des SMS

Ce déclencheur se produit lorsque l'utilisateur du bot a refusé de recevoir des SMS, soit manuellement soit par flux

#### La conversation est ouverte

Ce déclencheur se produit lorsque la conversation de l'utilisateur du bot est déplacée vers le dossier "Ouvert" dans le livechat

#### La conversation est en attente

Ce déclencheur se produit lorsque la conversation de l'utilisateur du bot est déplacée vers le dossier "En attente" dans le livechat

#### La conversation est terminée

Ce déclencheur se produit lorsque la conversation est déplacée vers le dossier "Terminé" dans le livechat

#### La conversation n’est pas valide

Ce déclencheur se produit lorsque la conversation est déplacée vers le dossier "Invalide" dans le livechat

#### Chat attribué à un agent

Ce déclencheur se produit lorsque l'utilisateur du bot est assigné à l'agent, soit manuellement soit par flux

#### Valeur du champ modifié

Ce déclencheur se produit lorsque la valeur d'un champ utilisateur spécifique est modifiée pour l'utilisateur du bot

#### Tag ajouté

Ce déclencheur se produit lorsque le tag est ajouté à l'utilisateur du bot

#### Tag supprimé

Ce déclencheur se produit lorsque le tag appliqué à l'utilisateur du bot est supprimé.

#### Inscrit à la séquence

Ce déclencheur se produit lorsque l'utilisateur du bot est abonné à une séquence.

#### Désabonnement d’une séquence

Ce déclencheur se produit lorsque l'utilisateur du bot est désabonné d'une séquence.

#### Déclencheur basé sur la date et l’heure

Ce déclencheur est utilisé lorsque vous voulez déclencher un sous-flux en fonction du temps avant ou après qu'un événement ait lieu. Vous ne pouvez utiliser que les champs de date et d'heure CUF dans ce déclencheur.

#### Commande payée

Ce déclencheur se produit lorsque l'utilisateur du bot a payé pour la commande

#### Commande commandée

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Commande en cours"

#### Commande en cours de traitement

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Traitement"

#### Commande expédiée

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Expédié"

#### Commande terminée

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Terminé"

#### Commande annulée

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Annulé"

#### Commande remboursée

Ce déclencheur se produit lorsque le statut de la commande est modifié en "Remboursé"

#### Commande Whatsapp reçue

Ce déclencheur se produit lorsque la commande est placée en utilisant le catalogue WhatsApp

#### Demande produit Whatsapp

Ce déclencheur se produit lorsque l'utilisateur du bot pose une question sur un produit en utilisant le catalogue WhatsApp

#### Parrainage Whatsapp

Ce déclencheur se produit lorsque l'utilisateur du bot est créé via une référence publicitaire WhatsApp

#### Parrainage publicitaire Facebook

Ce déclencheur se produit lorsque l'utilisateur du bot est créé via une référence publicitaire Facebook

#### Générateur de leads Facebook

Ce déclencheur se produit lorsque le lead est généré via une référence publicitaire Facebook

#### Le paiement de la facture Stripe à réussi

Ce déclencheur se produit lorsque le paiement de l'utilisateur du bot est réussi lors de l'utilisation de la passerelle de paiement Stripe native

#### Echec du paiement de la facture Stripe

Ce déclencheur se produit lorsque le paiement de l'utilisateur du bot a échoué lors de l'utilisation de la passerelle de paiement Stripe native

#### Webhook CRM v2

Ce déclencheur se produit lorsque le bot reçoit un webhook entrant de CRM V2. Vous pouvez spécifier le webhook dans le déclencheur. Nécessite que GHL/CRM soit connecté.

#### Webhook Shopify

Ce déclencheur se produit lorsque le bot reçoit un déclencheur de Shopify. Vous pouvez spécifier l'événement, comme la création de commande, le panier abandonné, etc.

#### Webhook Woocommerce

Ce déclencheur se produit lorsque le bot reçoit un déclencheur natif de Woocommerce. Vous pouvez spécifier l'événement, comme la création de commande, le panier abandonné, etc.

#### Chat invité terminé

Ce déclencheur se produit lorsque l'utilisateur du bot invité termine sa conversation. C'est utile pour le canal de webchat pour supprimer les utilisateurs invités

#### Calendly - Reservé

Ce déclencheur se produit lorsque l'utilisateur

du bot réserve un rendez-vous Calendly

#### Calendly - Reprogrammé

Ce déclencheur se produit lorsque l'utilisateur du bot reprogramme un rendez-vous Calendly

#### Calendly - Annulé

Ce déclencheur se produit lorsque l'utilisateur du bot annule un rendez-vous Calendly.

#### Mentionné dans la story

Ce déclencheur se produit lorsque quelqu'un mentionne l'utilisateur dans une histoire sur Instagram.

#### A répondu à la story

Ce déclencheur se produit lorsque quelqu'un répond à l'histoire de l'utilisateur.

#### A partagé un média

Ce déclencheur se produit lorsque quelqu'un partage le poste de l'utilisateur.
