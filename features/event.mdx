---
title: 'Evénements et rapports personnalisés dans LineChat'
---

Le suivi des événements et les rapports personnalisés dans LineChat permettent aux utilisateurs de suivre divers événements qui se produisent dans leurs assistant et de visualiser ces événements via des analyses sous forme d'événements personnalisés et de rapports personnalisés.

**Pour l'exemple nous allons tracker le nombre de clics sur un bouton**

### Création et utilisation d'événements personnalisés dans LineChat

Vous pouvez créer et utiliser des événements personnalisés dans LineChat en suivant les étapes suivantes :

1. Dans votre tableau de bord, cliquez sur le bot et entrez dans le constructeur de flux.
2. Sélectionnez l'onglet Contenu, puis sélectionnez les événements personnalisés.
    <img height="200" src="https://res.cloudinary.com/dfp1qo2jy/image/upload/v1718619915/SCR-20240617-leih_rx9yuu.png" />
3. Lors de la création d'un nouvel événement, vous devez fournir les paramètres suivants :
	* **Nom de l'événement** : utilisé pour nommer l'événement personnalisé.
	* **Description de l'événement** : utilisé pour décrire ce que l'événement personnalisé fait et suit.
	* **Noms de paramètres** : ces sont les noms des paramètres que vous souhaitez suivre, par exemple le produit, la catégorie de vente, le nom de la personne, etc.
    <img height="200" src="https://res.cloudinary.com/dfp1qo2jy/image/upload/v1718620073/SCR-20240617-lfdx_o7qfhf.png" />
<Info> Vous n'avez pas à remplir chaque paramètre pour que l'événement fonctionne, mais vous devez fournir au moins un paramètre pour suivre.</Info>
4. Après avoir créé un événement, entrez dans votre constructeur de flux et ajoutez l'action "Événement personnalisé" du bloc d'actions pour commencer à intégrer l'événement et à le suivre.

    L'evenement est créé, je peux donc l'utiliser dans le constructeur de flux :

### Visualisation des analyses pour les événements personnalisés

Pour visualiser les analyses pour les événements personnalisés, suivez les étapes suivantes :

1. Cliquez sur l'onglet Analytiques, sélectionnez les événements personnalisés pour afficher les analyses pour cet événement particulier.
2. Cliquez sur l'événement pour afficher les analyses détaillées de cet événement.

### Création et utilisation de rapports personnalisés dans LineChat

Vous pouvez créer et utiliser des rapports personnalisés dans LineChat en suivant les étapes suivantes :

1. Pour créer un rapport personnalisé, cliquez sur l'onglet Outils et sélectionnez la création de rapports personnalisés.
2. Entrez le nom du rapport et le type de rapport.
3. Cliquez sur l'icône d'engrenage pour commencer à modéliser votre rapport.

#### Modélisation de rapport de funnel

1. Cliquez sur les "Étapes de funnel" pour commencer à ajouter des événements personnalisés à votre rapport.
2. Cliquez sur "Ajouter une étape" et sélectionnez le type d'événement que vous souhaitez suivre.
3. Après avoir ajouté l'événement, cliquez sur les trois points pour commencer à personnaliser l'événement.
4. Configurez les événements que vous souhaitez suivre dans cet événement. Cliquez sur le bouton "Terminé".

#### Modélisation de rapport de pourcentage

1. Dans les rapports de pourcentage, vous pouvez comparer deux événements et déterminer comment ils se comportent par rapport à l'autre.
2. Pour ajouter les événements, cliquez sur le champ A et le champ B.
3. Pour modifier les graphiques et les tableaux, cliquez ici pour les personnaliser en fonction de vos besoins.

#### Formules

Les rapports de pourcentage permettent d'ajouter des formules pour calculer le pourcentage et l'événement approprié.

### Visualisation des analyses pour les rapports personnalisés

Pour visualiser les rapports personnalisés, suivez les étapes suivantes :

1. Pour visualiser les rapports personnalisés, cliquez sur l'onglet Analytiques et sélectionnez les rapports personnalisés.
