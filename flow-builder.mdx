title: 'Comprendre le constructeur de flux'
description: 'Comprenez et apprennez à utiliser le constructeur de flux'
---
Pour bien comprendre **comment marche le flow builder**, vous devez comprendre que votre **chatbot suis un fil de discussion que vous allez construire ici même, dans votre flow builder.**

Le flux de discussion est composé de différentes étapes reliés entre elles par des noeuds , les étapes sont accessible ici :

**Parmi ces étapes, il y'a :**

- Envoyer un message : permet d'envoyer un message comprenant du texte, une image etc...
- Question : très similaire au message, permet d'envoyer un message, et de récupérer la réponse de l'utilisateur sous forme de variable
- Action : permet de réaliser différentes actions qui traites les données du flux de discussion
- Condition (ici état) : permet de conditionner un choix de direction du flux de discussion
- Diviser : divise le flux de discussion
- Envoyer un mail : action d'envoie d'un mail
- Aller à : conduit directement le flux vers un bloc existant (pratique pour ne pas avoir des noeuds dans tous les sens)
- Commentaire : commentaire interne que vous êtes le seul à voir

Appuyez sur l'étape qui vous intéresse pour qu'elle apparaisse dans votre flux de discussion, **puis reliez les différentes étapes entres elles pour terminer le flux de discussion**.
