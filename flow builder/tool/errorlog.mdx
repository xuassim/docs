---
title: 'Journaux d'erreurs'
---
Les journaux d'erreurs peuvent vous aider à résoudre les bogues qui peuvent survenir à l'intérieur de votre chatbot. Cela peut vous faire gagner beaucoup de temps qui aurait été perdu à essayer de comprendre cela par vous-même autrement.

### Accès aux journaux d'erreurs

Pour accéder au journal d'erreurs, vous pouvez aller dans **Outils** dans le menu de gauche, puis sélectionner le premier onglet appelé **Journaux d'erreurs**.

### Informations fournies

À partir de là, vous aurez une vue d'ensemble complète de toutes les erreurs qui se sont produites dans le chatbot. Vous pourrez voir :

* La date de l'erreur
* Le type d'erreur
* L'utilisateur affecté
* Le sous-flux dans lequel cela s'est produit
* Le nœud qui contient l'erreur
* Plus d'informations

### Résolution des problèmes

Si vous cliquez sur le nœud du message d'erreur, cela vous emmènera directement au point où le problème a été enregistré. Cela accélérera considérablement la résolution des problèmes et vous fera gagner des heures que vous auriez autrement perdues à essayer de trouver le bogue.
