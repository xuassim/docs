---
title: 'Multilingue'
---
Vous avez la possibilité de proposer une expérience de chatbot multilingue avec LineChat.

### Comment ajouter une nouvelle langue

Pour ajouter une nouvelle langue, allez simplement dans **Outils** dans le menu de gauche, puis sélectionnez l'onglet **Langues multiples**.

#### Étapes à suivre

1. Appuyez simplement sur le bouton situé dans le coin supérieur droit nommé **+ Nouvelle langue**.
2. Une fois appuyé, une fenêtre contextuelle apparaîtra où vous pourrez sélectionner une nouvelle langue.
